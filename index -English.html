Scientific Projects




Tablets Tracker

Computergestützte Methoden zur Vorhersage der Endqualität eines pharmazeutischen Produkts auf der Grundlage seiner Schalleigenschaften.

Tablets Tracker

In diesem Projekt haben ich und mein Forschungsteam unter der Leitung von Prof. Dr. rer. nat. Ulrich Odefey und Prof. Dr. Miriam Pein-Hackelbusch erfolgreich Methoden zur Überwachung der physikalischen Eigenschaften eines pharmazeutischen Produkts (Textur, Dichte, etc.) entwickelt. Das Herzstück des Projekts ist unser Algorithmus, der spezielle Schallwellen aufnimmt, die sich durch das Produkt ausbreiten. Der Algorithmus führt einen Signalverarbeitungsprozess durch, um Merkmale zu extrahieren, die dann an unser trainiertes Machine Learning Modell weitergeleitet werden, das auf der Grundlage dieser Merkmale die Endqualität vorhersagt.

Bilirubin Scanner

Computer Vision Algorithmus zur Bewertung von Bilirubin Blutwerten anhand digitaler Bildern

Bilirubin Scanner

In diesem Projekt habe ich unter der Leitung von Prof. Dr. rer. nat. Helene Dörksen und dem inIT-Forschungsinstitut erfolgreich einen Algorithmus entwickelt, der eine Bildverarbeitung auf digitalen Bildern von Augen unter üblichen Lichtbedingungen durchführt, um Merkmale zu extrahieren, die dann an ein von mir trainiertes Machine Learning Model weitergegeben werden, um anhand dieser Merkmale den genauen Blutbilirubinspiegel der Person, zu der das Bild gehört, zu approximieren.

Sclera Detector

Ein Verfahren zur Segmentierung der Sklera aus Bildern menschlicher Gesichter

Sclera Detector

In diesem Projekt habe ich unter der Leitung von Prof. Dr. rer. nat. Helene Dörksen erfolgreich einen Algorithmus entwickelt, der sowohl Deep Learning für die Augenerkennung einsetzt als auch verschiedene Bildverarbeitungstechniken, um die Pixel zu erkennen und zu isolieren, die dem Augenweiß (d.h. der Sklera) auf einem beliebigen Bild eines menschlichen Gesichts entsprechen. Dies ist ein wichtiger Schritt für verschiedene Bildanalyseanwendungen, die die Sklera betreffen.

Raspberry Eye

Kameraüberwachungssystem für industrielle Anwendungen

Raspberry Eye


Verwendete Technologien: Java (Springboot, Hibernate, usw), Python (OpenCV), SQL und Raspberry Pi

In diesem Projekt habe ich eine plattformunabhängige Software entwickelt, die die Interaktion mit einer Raspberry Pi-basierten Kamera ermöglicht. Sie ermöglicht einen lokalen Live-Stream der Eingabe sowie einen Online-Live-Stream. Für letzteres interagiert die Software mit einem Server. Ich habe die serverseitige Anwendung unter anderem mit dem Java-basierten Framework "Spring boot" entwickelt, um eine optimale Geschwindigkeit zu erreichen.


Ich habe diese Software im Rahmen meiner Tätigkeit am Institut für Industrielle Informationstechnik (inIT) entwickelt.
